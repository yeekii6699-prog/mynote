# Codex 记忆索引

> “模块化记忆库”已启用：上线前先看索引，再进入对应模块；需要更新内容时直接编辑相应文件。

## 快速入口
| 模块 | 路径 | 作用 |
| --- | --- | --- |
| 01. 个人画像与 Summary | `modules/profile.md` | 人设、联系方式、求职意向、Summary |
| 02. 技能矩阵 | `modules/skills.md` | 专业技能与长期维护的能力清单 |
| 03. 代表项目 | `modules/projects.md` | 展示代表性的 AI / RPA 应用案例 |
| 04. 工作经历 | `modules/work_experience.md` | 各阶段职位职责与待补信息 |
| 05. 目标 & 创始人画像 | `modules/goals.md` | Core Objective / Drive / INTJ 画像 |
| 06. 交互指令 | `modules/directives.md` | 语气、人设、沟通 SOP |
| 07. 门店数字化方案 | `modules/shop_system.md` | 去 SaaS 化架构、流程与路线图 |

## 使用说明
1. **上线前复习**：至少浏览 Profile、Directives、Shop System 三个模块，保证人设与业务上下文同步。
2. **更新记忆**：若某模块有新增内容，直接在对应文件中维护并标注日期；若新增主题，请同步更新本索引和 `ARCHITECTURE.md`。
3. **写作习惯**：保持简体中文 + 口语化 + Emoji（✨, 🥺, 💪, ❤️, 🐱），结构清晰，贴合 INTJ 的逻辑偏好。
