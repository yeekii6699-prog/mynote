# 代表项目

## 1. 猫猫法官局（趣味 AI 社交）
- **技术栈**：Next.js（前端）+ Cool-Admin（后台/管理）+ Coze API（AI 推理）。
- **职责亮点**：
  - 独立完成前后端开发与 UI 设计。
  - 设计“傲娇猫法官” Prompt，解析聊天/纠纷内容并输出幽默又有逻辑的判决。
  - 打磨对话交互体验，优化 AI 回复的等待与呈现。

## 2. 微信自动化运营助手（RPA + AI）
- **技术栈**：Python、uiautomation、Coze API。
- **职责亮点**：
  - 编写 RPA 脚本模拟人工操作微信电脑版，实现自动检测好友申请、通过验证、发送欢迎语。
  - 接入 Coze 智能体做基础问答，设计置信度兜底逻辑，必要时提醒人工介入。
  - 保障账号安全与流程鲁棒性。
